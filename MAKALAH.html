BAB I
PENDAHULUAN

1.1 Latar Belakang

Manajemen keuangan pribadi merupakan aspek penting dalam kehidupan modern yang seringkali diabaikan. Banyak individu mengalami kesulitan dalam melacak pengeluaran, mengatur anggaran, dan membuat keputusan finansial yang tepat. Dengan perkembangan teknologi Progressive Web Application (PWA) dan Artificial Intelligence (AI), kini dimungkinkan untuk menciptakan solusi yang dapat membantu pengguna mengelola keuangan mereka secara lebih efektif dan efisien.

DompetCerdas AI adalah aplikasi berbasis web yang dirancang untuk membantu pengguna dalam mencatat transaksi keuangan, memvisualisasikan data pengeluaran dan pemasukan, serta mendapatkan rekomendasi finansial yang dipersonalisasi melalui teknologi AI. Aplikasi ini dapat di-install pada perangkat mobile maupun desktop layaknya aplikasi native, namun tetap mempertahankan kemudahan akses melalui browser.

1.2 Rumusan Masalah

  1. Bagaimana merancang sistem manajemen keuangan pribadi yang user-friendly dan accessible?
  2. Bagaimana mengintegrasikan teknologi AI untuk memberikan insight dan rekomendasi finansial?
  3. Bagaimana mengimplementasikan PWA agar aplikasi dapat berfungsi offline dan installable?

1.3 Tujuan

  1. Mengembangkan aplikasi web untuk manajemen keuangan pribadi dengan antarmuka yang intuitif
  2. Mengimplementasikan fitur AI advisor menggunakan Google Gemini API
  3. Menerapkan teknologi PWA untuk meningkatkan user experience dan accessibility

1.4 Manfaat

Manfaat Teoritis:

  • Mendemonstrasikan implementasi PWA dalam aplikasi finansial
  • Menunjukkan integrasi AI dalam pengambilan keputusan finansial
  • Mempelajari best practices dalam web development modern

Manfaat Praktis:

  • Membantu pengguna melacak pengeluaran dan pemasukan secara real-time
  • Memberikan visualisasi data keuangan yang mudah dipahami
  • Menyediakan rekomendasi finansial yang personal dan actionable

2.2 Arsitektur Sistem

Arsitektur sistem DompetCerdas AI terdiri dari beberapa komponen utama:

  • User Interface Layer: Komponen React untuk Dashboard, Transaction Form, Transaction List, dan AI Advisor
  • Business Logic Layer: State management menggunakan React Hooks (useState, useEffect)
  • Data Layer: LocalStorage untuk persistensi data transaksi
  • Service Layer: Gemini API service untuk AI-powered recommendations
  • PWA Layer: Service Worker untuk offline caching dan installability

2.4 Penjelasan Flowchart

2.4.1 Alur Utama Aplikasi

1. Inisialisasi: Saat aplikasi dimulai, sistem melakukan load data transaksi dari LocalStorage menggunakan useEffect hook. Jika data tersedia, aplikasi akan menampilkan dashboard dengan visualisasi lengkap.

2. Navigasi Menu: User dapat memilih 3 menu utama (Dashboard, Transaksi, AI Advisor) melalui sidebar (desktop) atau bottom navigation (mobile). State view akan diupdate sesuai pilihan user.

3. Tambah Transaksi: Ketika user menambahkan transaksi baru, sistem melakukan validasi input, generate UUID menggunakan crypto.randomUUID(), lalu menyimpan data ke state dan LocalStorage secara bersamaan. Visualisasi dashboard akan otomatis terupdate berkat React reactivity.

2.4.2 AI Advisor Flow

1. Data Preparation: Sistem mengumpulkan semua transaksi dari state dan memformat data menjadi struktur yang dapat dibaca oleh AI (JSON ke text summary).

2. API Call: Request dikirim ke Google Gemini API melalui service layer (geminiService.ts) dengan prompt yang berisi konteks transaksi user dan instruksi untuk memberikan rekomendasi finansial.

3. Response Handling: Response dari AI di-parse menggunakan react-markdown untuk menampilkan format yang lebih readable dengan dukungan bold, italic, list, dan code blocks.

4. Error Handling: Jika API gagal (network error, API key invalid, dll), sistem menampilkan error message yang user-friendly dan memberikan saran troubleshooting.

A.2 Gemini AI Service (geminiService.ts)

import { GoogleGenerativeAI } from '@google/genai';

const API_KEY = process.env.GEMINI_API_KEY || '';
const genAI = new GoogleGenerativeAI(API_KEY);

export async function getFinancialAdvice(
  transactions: any[]
): Promise<string> {
  try {
    const model = genAI.getGenerativeModel({ 
      model: 'gemini-1.5-flash' 
    });

    const totalIncome = transactions
      .filter(t => t.type === 'income')
      .reduce((sum, t) => sum + t.amount, 0);
      
    const totalExpense = transactions
      .filter(t => t.type === 'expense')
      .reduce((sum, t) => sum + t.amount, 0);

    const prompt = `
      Analisis data keuangan berikut dan berikan saran:
      
      Total Pemasukan: Rp ${totalIncome.toLocaleString('id-ID')}
      Total Pengeluaran: Rp ${totalExpense.toLocaleString('id-ID')}
      Saldo: Rp ${(totalIncome - totalExpense).toLocaleString('id-ID')}
      
      Detail Transaksi:
      ${transactions.map(t => 
        `- ${t.type}: ${t.description} (Rp ${t.amount.toLocaleString('id-ID')})`
      ).join('\n')}
      
      Berikan rekomendasi finansial yang spesifik dan actionable.
    `;

    const result = await model.generateContent(prompt);
    const response = await result.response;
    return response.text();
    
  } catch (error) {
    console.error('Gemini API Error:', error);
    throw error;
  }
}

A.3 Service Worker (sw.js)

const CACHE_NAME = 'dompetcerdas-v1';
const STATIC_CACHE = 'dompetcerdas-static-v1';
const DYNAMIC_CACHE = 'dompetcerdas-dynamic-v1';

const STATIC_ASSETS = [
  '/',
  '/index.html',
  '/index.css',
  '/icon-192.png',
  '/icon-512.png',
  '/manifest.json'
];

// Install - cache static assets
self.addEventListener('install', (event) => {
  event.waitUntil(
    caches.open(STATIC_CACHE).then((cache) => {
      return cache.addAll(STATIC_ASSETS);
    })
  );
  self.skipWaiting();
});

// Activate - clean old caches
self.addEventListener('activate', (event) => {
  event.waitUntil(
    caches.keys().then((cacheNames) => {
      return Promise.all(
        cacheNames.map((cache) => {
          if (cache !== STATIC_CACHE && cache !== DYNAMIC_CACHE) {
            return caches.delete(cache);
          }
        })
      );
    })
  );
  self.clients.claim();
});

// Fetch - serve from cache, fallback to network
self.addEventListener('fetch', (event) => {
  if (!event.request.url.startsWith(self.location.origin)) {
    return;
  }

  event.respondWith(
    caches.match(event.request).then((cachedResponse) => {
      if (cachedResponse) {
        return cachedResponse;
      }

      return fetch(event.request).then((response) => {
        if (!response || response.status !== 200) {
          return response;
        }

        const responseToCache = response.clone();
        caches.open(DYNAMIC_CACHE).then((cache) => {
          cache.put(event.request, responseToCache);
        });

        return response;
      });
    }).catch(() => {
      return caches.match('/index.html');
    })
  );
});

BAB III
PENUTUP

3.1 Kesimpulan

Berdasarkan pembahasan yang telah diuraikan, dapat disimpulkan bahwa:

  1. Implementasi PWA berhasil dilakukan dengan menggunakan Service Worker dan Web App Manifest, memungkinkan aplikasi untuk di-install di perangkat mobile dan desktop serta berfungsi secara offline.
  2. Integrasi Google Gemini AI memberikan nilai tambah signifikan dengan menyediakan rekomendasi finansial yang personal dan contextual berdasarkan pola transaksi pengguna.
  3. React + Vite sebagai tech stack terbukti efektif untuk development yang cepat dengan performa optimal. Hot Module Replacement (HMR) mempercepat development cycle.
  4. LocalStorage sebagai data persistence cukup untuk aplikasi personal finance scale kecil, dengan kelebihan privacy (data tidak ke server) dan simplicity (tidak perlu backend).
  5. Responsive design memastikan aplikasi dapat digunakan dengan baik di berbagai ukuran layar, dari mobile hingga desktop.

3.2 Saran

Untuk pengembangan lebih lanjut, disarankan:

  1. Backend Integration: Menambahkan backend server (Node.js + MongoDB) untuk sinkronisasi data antar device dan backup cloud.
  2. Advanced Analytics: Implementasi machine learning untuk prediksi pengeluaran bulanan dan anomaly detection.
  3. Multi-currency Support: Menambahkan dukungan untuk berbagai mata uang dengan exchange rate real-time.
  4. Export Features: Fitur export data ke CSV/PDF untuk keperluan reporting dan tax filing.
  5. Budgeting Module: Menambahkan fitur budget planning dengan alert ketika mendekati atau melebihi budget.
  6. Push Notifications: Reminder untuk input transaksi harian dan alert untuk bill payment.
  7. Collaborative Features: Shared budget untuk keluarga atau roommates dengan permission management.

3.3 Penutup

DompetCerdas AI mendemonstrasikan bagaimana teknologi modern seperti PWA dan AI dapat dikombinasikan untuk menciptakan solusi praktis dalam manajemen keuangan pribadi. Dengan pendekatan user-centric design dan teknologi yang accessible, aplikasi ini dapat membantu siapa saja untuk mengelola keuangan mereka dengan lebih baik. Semoga aplikasi ini dapat memberikan manfaat nyata bagi penggunanya dan menjadi starting point untuk inovasi lebih lanjut di bidang financial technology.
